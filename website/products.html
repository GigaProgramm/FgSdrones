Project Overview

Web Simulator

Designed for quick testing of drone programs without using additional modules. Allows you to prototype and debug your drone code in a virtual environment before deploying to real hardware.

Unity Game

Provides the ability to assemble drones, program them and use them in various scenarios. Features an open-world environment where you can test your drones in challenging situations.

IDE and Code Editor

Allows you to create programs for drones and download them directly to devices. Supports installation of custom modules to expand drone functionality. The list of supported modules can be found in the project documentation.

This project is aimed at a wide audience, including beginners, teachers, organizers and participants of robotics and programming competitions. It provides convenient tools for learning the basics of drone programming, creating your own projects and participating in competitions of various levels.

The intuitive interface and simplified programming language make it accessible to users with no prior experience, while the advanced features satisfy the needs of more experienced developers.
